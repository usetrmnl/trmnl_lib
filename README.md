# TRMNL library
A link library to easily add TRMNL functionality to any project.<br>

To create the trmnl-display application, type the following:

cd linux/examples/trmnl_display

For MacOS, type the following:
./build_macos.sh

For Linux (including the Raspberry Pi(s)), type the following:
./build_linux.sh

When the build finishes, to run trmnl_display simply type:
./trmnl_display
